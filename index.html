A Novel Challenge for long-context language models

Welcome to Nocha Leaderboard! Nocha tests the capabilities of long-context models to verify claims about fictional books. The dataset consists of true/false narrative minimal pairs about the same event or character. The model must verify both claims in a pair to be awarded 1 point. We report the accuracy (%) along with the counts of correct answers and total attempts. Since models differ in how much text they can process, the total number of pairs (attempts) processed by each model will vary. You can select models below to show the accuracies for a common set of pairs attempted by these models.

Example of our dataset:


{
  "claim 1"
  "claim 2"
  "book"
  etc
}

Code and sample data available on GitHub.
